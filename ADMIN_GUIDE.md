# 🌸 Linden Blossom Admin Panel

## Доступ к админ панели

### URL админ панели
```
https://linden-blossom-api.privatetwe7ve.workers.dev/admin
```

### Учетные данные

**Логин:** `admin`  
**Пароль:** Смотрите переменную `ADMIN_PASSWORD`, которую вы установили командой:

```bash
wrangler secret put ADMIN_PASSWORD
```

При запросе пароля введите ваше сохраненное значение.

---

## 📊 Возможности админ панели

### 1. **Управление заказами**
- Просмотр всех заказов с деталями
- Изменение статуса заказа:
  - `new` — новый заказ
  - `confirmed` — подтвержден
  - `done` — готов
  - `cancelled` — отменен

### 2. **Управление букетами**
- Редактирование JSON каталога букетов
- Изменение цен, описаний и других параметров
- Кнопка "Format" для форматирования JSON
- Кнопка "Save" для сохранения изменений

---

## 🔐 Аутентификация

Админ панель использует **Basic Authentication**:

```
Authorization: Basic base64(admin:ваш_пароль)
```

Пароль автоматически кодируется браузером при входе.

---

## 📝 Структура зашифрованных данных

### Переменные окружения (в wrangler.toml)
```toml
[vars]
ADMIN_USER   = "admin"
CORS_ORIGIN  = "https://linden-blossom.pages.dev"
```

### Секреты (через `wrangler secret`)
```bash
wrangler secret put ADMIN_PASSWORD  # Сохраняет пароль на сервере
```

---

## 🔗 API Endpoints

| Метод | Endpoint | Описание |
|-------|----------|---------|
| GET   | `/admin` | HTML админ панели |
| GET   | `/admin/orders` | Список заказов (JSON) |
| PATCH | `/admin/orders/:id` | Изменить статус заказа |

Все запросы требуют **Basic Auth** заголовок.

---

## 💾 Локальное хранилище

После первого входа в панель ваши учетные данные сохраняются в `localStorage` браузера:
- `admin_user` — логин
- `admin_pass` — пароль

При перезагрузке страницы вы автоматически войдете, если они там хранятся.

---

## 🚀 Развертывание изменений

После редактирования букетов в админ панели:

1. Нажмите кнопку **"Save"**
2. Данные сохраняются в KV хранилище (`BOUQUETS_STORE`)
3. Изменения видны на сайте при следующей загрузке каталога

---

## ⚠️ Безопасность

- **Не** делитесь пароль на публичных местах
- Используйте сильный пароль (`wrangler secret put ADMIN_PASSWORD`)
- Админ панель защищена Basic Auth
- CORS ограничен на `https://linden-blossom.pages.dev`

---
